# Prompt — generate a self-contained Three.js world.html

This directory holds the only prompt in the repo.

- [`prompt.md`](prompt.md) — copy and paste this entire file into a model.
  Save the raw output as `world.html`, open it in a browser, and look at
  it. Run the same prompt across different models and compare the
  resulting HTML files side by side.

The model never sees `outputs/create.html` or any other reference file.
That file is a hand-tuned ideal world for *you* — to judge outputs and to
keep refining `prompt.md`. The prompt is pure natural language: it
describes the *kind* of world to generate. Results may differ in scale and
layout; they can be better. What matters is ecological relationships,
climate logic, and the voxel visual style — not cloning measurements.
